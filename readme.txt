================================================================================
# 梱包サイズ推定アプリ
================================================================================

ホビー商品（鉄道模型・フィギュア・プラモデル等）の過去梱包実績データをもとに、
AIが類似商品の梱包サイズ・梱包重量を自動推定する Webアプリ

---

## 要件定義

### 背景・目的

ECサイトにおいて、登録商品の梱包サイズ・梱包重量を事前に把握することは正確な送料徴収の観点で重要
本アプリは過去の梱包実績データとAIを組み合わせることで、なるべく高い精度で梱包サイズを推定できる環境を提供する

### 機能要件

- ユーザーが商品情報（商品名・メーカー・カテゴリ等）を入力すると梱包サイズを推定できる
- RAGあり推定（過去実績データ参照）と RAGなし推定（AI単独）の両方を表示し比較できる
- 類似スコアに基づいてどちらの推定を信頼すべきか推奨を提示する
- ログイン認証によりアクセスを制限できる
- 管理者がユーザーの追加・削除・パスワード変更を行える

### 非機能要件

- RAGあり・RAGなしのLLM呼び出しを並列実行し、応答時間を短縮する
- パスワードはbcryptでハッシュ化して保存し、平文では保存しない

## 動作イメージ

1. ログイン画面でデモアカウントでログイン
2. 商品情報（メーカー・カテゴリ・商品名など）を入力して「梱包サイズを調べる」をクリック
3. RAGあり推定・RAGなし推定の結果が並べて表示される
4. 推奨判定（★推奨バッジ）で信頼度の高い推定を確認

---

## 機能詳細

### 1) 梱包サイズ推定

| ファイル 					| 役割									|
|---------------------------|---------------------------------------|
| pages/1_梱包サイズ推定.py	| 検索UI・結果表示						|
| utils/pinecone_client.py	| Pinecone検索・LLM推定のコアロジック		|
| config.py					| メーカー・カテゴリ・シリーズの選択肢定義 など	|


【検索フロー】

1. 入力		JANコード・商品名・メーカー・カテゴリ・サブカテゴリ・シリーズを入力（3項目以上必須）
2. 類似検索	入力テキストをベクトル化（OpenAI Embeddings）し Pinecone でコサイン類似度検索
			メーカー・カテゴリ・シリーズでメタデータフィルタをかけ、件数不足時は段階的にフィルタを緩めてフォールバック
3. 推定		RAGあり（類似商品の実績データをプロンプトに渡してLLMが推定）と RAGなし（LLMの学習知識のみで推定）を並列実行
4. 推奨判定	類似スコアに基づき RAGあり・RAGなしどちらを信頼すべきか自動判定してバナー表示


【結果表示】

・RAGあり・RAGなしの推定値（縦・横・高さ・重量・信頼度）を2カラムで並べて比較
・推奨側に「★ 推奨」バッジを表示
・フィルタの適用状況（カテゴリ絞り込み済み / フォールバックした場合は警告）をキャプションで表示


### 2) ユーザー管理

| ファイル 				| 役割 													|
|-----------------------|-------------------------------------------------------|
| utils/database.py		| SQLite（users.db）の読み書き・パスワードのbcryptハッシュ管理	|
| utils/auth.py			| session_stateを使ったログイン状態の管理					|
| Home.py				| ログイン画面（エントリーポイント）							|
| pages/2_ユーザー管理.py	| 管理者専用のユーザー管理ページ								|


【認証フロー】
1. 起動時		init_db() が実行され、users.db がなければ自動生成。管理者アカウント（admin）も初回のみ自動作成
2. ログイン		ユーザー名・パスワードを入力 → bcryptで照合 → 成功時に session_state にユーザー情報を保存
3. ページ保護		各ページ先頭で require_login() または require_admin() を呼ぶだけでアクセス制限
				未ログインは st.stop() で処理を停止
4. ログアウト		session_state.clear() でセッションを破棄


【権限】
・一般ユーザー	梱包サイズ検索ページのみ利用可
・管理者			ユーザーの追加・削除・パスワード変更が可能。最後の管理者は削除不可

## Pineconeのデータ構造

  {
    "id": "4543736221587",					    // JANコード
    "values": <search_text を embed した 1536 次元ベクトル>,
    "metadata": {
      "jan":             "4543736221587", 		// JANコード
      "product_name_ja": "DD51 1000番台 JR仕様",	// 商品名
      "series":          "Nゲージ",				// シリーズ
      "manufacturer":    "KATO",				// メーカー
      "category_2":      "Model Trains",		// カテゴリ2
      "category_3":      "Train Parts",			// カテゴリ3
      "shipping_length": 31.0,					// 梱包サイズ_縦
      "shipping_width":  20.0,					// 梱包サイズ_横
      "shipping_height": 10.0,					// 梱包サイズ_高さ
      "shipping_weight": 0.5					// 梱包重量
    }
  }

　注) search_text = ベクトル化対象の下記の値を連結したもの (JANコードは含まない）
	商品名
	シリーズ
	メーカー
	カテゴリ2
	カテゴリ3
